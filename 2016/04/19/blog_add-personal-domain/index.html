How to build a personal blog using GitHub Pages and Hexo - 3 - Add Domain

Posted on 2016-04-19   |   In Hexo   |  

开启新世界的大门

First, make sure that you can visit you own blog by using the Github Page url, like your-github-account-name.github.io.

The next step is going to link to your own domain, then you can visit your blog like www.unclegarden.com.

Hexo - add CNAME

We need to create a new file - CNAME. Put this file in our Hexo project under source folder.

../Hexo/source ls
CNAME _posts

In the CNAME file, what you should write is only your own domain name.

cat CNAME
unclegarden.com

Then, you just need to do the hexo deploy, the CNAME file will be added.

GoDaddy

After you buying your own domain, just go to your domain detals and let us start to do the configuration.
