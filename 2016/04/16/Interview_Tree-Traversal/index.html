关于树的题目Leetcode有很多，此篇整理关于Binary Tree遍历的几种方法。讨论pre-Order, in-Order, post-Order, level-Order。根据不同的顺序，pre-Order是中-左-右的顺序，in-Order是左-中-右的顺序，post-Order是左-右-中的顺序，level-Order是按照层次遍历。

Level Order

102, 和 107 两题的唯一区别就是一个是要你从上往下按照层次排列每层的元素，一个则是需要从下往上。

根据解题套路来

  1. input。root，然后Leetcode给出了TreeNode的数据结构（发散下思维，就是面试的时候，有些follow up，或者是开始答题之前，数据结构问清楚，可以自己设计）
  2. 报错判断。对于TreeNode，我们一般处理是会取值.val，或者取左右子树.left, .right。所以需要对current node是不是null进行判断。
  3. 入口。对于level order，我们肯定需要记录一下层，然后才能准确的对节点进行操作。入口当然就是root，第0层。同时，这个层的记录，也是用来判断是否需要增加一个新的list来添加新一层的节点值。
  4. 出口。出口就是当节点是null的时候，就是到了null叶子，返回。
  5. Test Case。
  6. 空间复杂度，时间复杂度。？，O（n）
  7. Follow up。递归之后，一般考虑迭代行不行。
public class Solution {
    public List<List<Integer>> levelOrder(TreeNode root) {
        List<List<Integer>> result = new ArrayList<>();
        helper(root, result, 0);
        return result;
    }

        private void helper(TreeNode root, List<List<Integer>> result, Integer level) {
        if (root == null) return;
        if (result.size() == level) result.add(new ArrayList<Integer>());
        result.get(level).add(root.val);
        helper(root.left, result, level+1);
        helper(root.right, result, level+1);
    }    
}

迭代 - 空间复杂度，时间复杂度

public class Solution {
    public List<List<Integer>> levelOrder(TreeNode root) {
        List<List<Integer>> result = new ArrayList<>();
        Queue<TreeNode> queue = new LinkedList<>();

        queue.add(root);

        while(!queue.isEmpty()){
            List<Integer> level = new ArrayList<>();
            int len = queue.size();
            for(int i = 0; i < len; i++){
                TreeNode node = queue.poll();
                if(node != null){
                    level.add(node.val);
                    queue.add(node.left);
                    queue.add(node.right);
                }
            }

            if(!level.isEmpty()){
                result.add(level);
            }
        }
        return result;
    }
}

public class Solution {
    public List<Integer> inorderTraversal(TreeNode root) {
        List<Integer> result = new ArrayList<>();
        inorderTraversalHelper(result, root);
        return result;
    }

    private void inorderTraversalHelper(List<Integer> result, TreeNode root){
        if(root == null) return;
        inorderTraversalHelper(result, root.left);
        result.add(root.val);
        inorderTraversalHelper(result, root.right);
    }
}

迭代 - 空间复杂度，时间复杂度

public class Solution {
    public List<Integer> inorderTraversal(TreeNode root) {
        List<Integer> result = new ArrayList<>();
        if(root == null) return result;

        Stack<TreeNode> s = new Stack<>();
        # 先存入root节点。
        s.push(root);
        # in order， left-root-right, 先找到root的left节点
        TreeNode current = root.left;
        while(!s.isEmpty() || current != null){
            # 这个if判断为了找到最左的子节点。
            if(current != null){
                s.push(current);
                current = current.left;
            }
            else{
                # 首先，因为上一个if找到最左的子节点，stack pop出的是整棵树的最左的节点，如果此节点有右子树，那么此节点相当于也是一个current root。
                current = s.peek();
                result.add(current.val);
                s.pop();
                current = current.right;
            }
            # 重复过程。找到每个current root最左的子节点，。
        }
        return result;
    }
}

public class Solution {
    public List<Integer> preorderTraversal(TreeNode root) {
        List<Integer> result = new ArrayList<>();
        # 如果root是null，就直接返回空的list
        if(root == null) return result;
        # 增加一个stack用于记录
        Stack<TreeNode> s = new Stack<>();
        # pre order，root-left-right。先存入root节点。
        s.push(root);

        while(!s.isEmpty()){
            # 每次直接pop出当前节点。并添加到结果中。
            TreeNode current = s.pop();
            result.add(current.val);
            # 因为stack， FILO。又因为root-left-right。所以先存入right进stack，再存入left。
            # 然后当stack pop的时候，会先得到left，再right。符合需求。
            if(current.right != null) s.push(current.right);
            if(current.left != null) s.push(current.left);
        }
        return result;
    }
}

Postorder

  1. 入口。
  2. 出口。出口就是当节点是null的时候，就是到了null叶子，返回。
  3. 空间复杂度，时间复杂度。

递归

public class Solution {
    public List<Integer> postorderTraversal(TreeNode root) {
        List<Integer> result = new ArrayList<>();
        postorderTraversalHelper(result, root);
        return result;
    }

    private void postorderTraversalHelper(List<Integer> result, TreeNode root){
        if(root == null) return;
        postorderTraversalHelper(result, root.left);
        postorderTraversalHelper(result, root.right);
        result.add(root.val);
    }
}

迭代 - 空间复杂度，时间复杂度

public class Solution {
    public List<Integer> postorderTraversal(TreeNode root) {
        List<Integer> result = new ArrayList<>();
        if(root == null) return result;

        Stack<TreeNode> s = new Stack<>();
        # 先放入root
        s.push(root);
        # 需要一个record的节点，用来记录之前被访问的node
        TreeNode record = null;
        # 只要判断stack，因为stack空的情况就是root也被输出了，就是整棵树遍历完了。
        while(!s.isEmpty()){
            TreeNode current = s.peek();
            # 三个条件。1. 根节点的时候root，2. 还在往左找，record.left == current，3. 还在往右找，record.right == current
            if(record == null || record.left == current || record.right == current){
                # 一直往左找，左节点加入stack
                if(current.left != null){
                       s.push(current.left);
                }
                # 如果没有左节点，当前层的右节点加入stack
                else if(current.right != null){
                       s.push(current.right);
                }
                # 没有左节点，并且没有右节点，根据post order，left-right-root。当前node就是一个当前的root，pop出，并加入结果list
                else{
                    s.pop();
                    result.add(current.val);
                }
            }
            # 当pop出一个左节点的同时，需要double check当前节点有没有右节点，有就加入stack继续以上操作。没有就可以直接pop当前节点，因为它就是一个当前的root。
            else if(record == current.left){
                if(current.right != null){
                    s.push(current.right);
                }else{
                    s.pop();
                    result.add(current.val);
                }
            }
            # 根据上一条，如果pop出的是右节点，那么根据post order的规则，当前节点就是一个当前的root，可以直接pop出，加入结果list。
            else if(record == current.right){
                s.pop();
                result.add(current.val);
            }
            # 记得记录之前的一个节点。
            record = current;
        }
        return result;
    }
}
