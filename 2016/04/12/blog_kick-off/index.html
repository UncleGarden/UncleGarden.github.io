How to build a personal blog using GitHub Pages and Hexo - 1

Posted on 2016-04-12   |   In Hexo   |  

五十岁的老司机我笑脸扬啊

Pre-requisite

Please install node.js and git in your local machine. And go to Github to create your own account.

  • node.js

  • git

  • Github

  • GitHubPages

Create the personal blog repository in Github. The name of the new repository should be like username.github.io.

Follow the link to setup the ssh key Generate ssh key

ssh -T git@github.com

Follow the link to setup the index page for your own website Github Page - Hello World. Go to http://username.github.io, you can see “Hello World”.

git clone your/new/repo

cd new/repo
echo "Hello World" > index.html

git add .
git commit -m ''
git push

Setup Hexo

Hexo install

cd path/to/your/workspace
npm install hexo-cli -g

Hexo deploy (local)

cd path/to/your/workspace
hexo init Hexo

Install the dependency.

cd Hexo
npm install

Then, we can generate the blog and start the server to see the original blog.

hexo generate
hexo server

http://localhost:4000

Hexo deploy to Github

Update _config.yml

Update the Hexo/_config.yml, link to your own github repo.

Deploy

hexo clean
hexo generate
hexo deploy

you may meet this error:

First, update the depoly type to git (in the above _config.yml)

and run the following code

npm install hexo-deployer-git --save

Then, run the deploy code again.

You are all set! Go to http://your_user_name.github.io
